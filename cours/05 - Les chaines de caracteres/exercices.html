Les chaînes de caractères

Pour répondre à ces exercices, tapez votre code dans le fichier exercices.js.
Si vous êtes bloqué, venez demander de l'aide sur Slack.

Objectifs

  • Utiliser l'instruction console.log() pour afficher des chaînes de caractères.
  • Apprendre à échapper des caractères avec \.
  • Apprendre à concaténer plusieurs chaînes de caractères avec l'opérateur +.

Affichez la phrase JavaScript est facile.

Exercice 1

Exercice 2
